# Task 4 - Security Enhancements

## Overview
This task focuses on improving the security of the CRUD application developed in the previous tasks by implementing secure coding practices, input validation, and role-based access control.

## Features Implemented
- Prepared Statements using MySQLi
- Server-side Form Validation
- Client-side Form Validation
- User Roles (Admin & Editor)
- Role-Based Access Control
- Improved Authentication Security
- Protection against SQL Injection

## Technologies Used
- PHP
- MySQL
- MySQLi
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- XAMPP

## Security Improvements
- All database queries use prepared statements.
- User input is validated on both client and server sides.
- Passwords are securely stored using password hashing.
- SQL Injection vulnerabilities are mitigated.
- User permissions are controlled using roles.

## Project Structure
- Task 2: CRUD Application
- Task 3: Search, Pagination & UI Enhancements
- Task 4: Security Enhancements
